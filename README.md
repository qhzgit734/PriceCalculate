## 一、说明

## 二、风管附件
1. 风管附件计算方法主要参考浙江信息价2022.05
2. 风管附件模块中圆形尺寸，在A、B中同时输入直径
3. 热镀锌板6200元/t,其价格影响系数为:每涨(跌)300元/t时,价格上下浮动3%
4. 铝合金风口型材按上海期货铝锭22000元/t；其价格影响系数为:每涨(跌)1500元/t时,价格上下浮动5%
## 三、焊接钢管
套管=电缆保护管
## 四、镀锌钢管
1. 指水管
2. 热镀锌=热浸锌=热浸锌加厚，KBG/JDG为薄壁镀锌钢管＜水
